# Graphical Config Files

Just a repo with configurations files located at `$HOME/.config`.
Feel free to pull request to complete [TODO list](#todo).

## Dependencies

- i3wm
- picom
- polybar
- [ixwindow](https://github.com/mattthhh/ixwindow) (copy build to $PATH)
- dmenu_run_history
